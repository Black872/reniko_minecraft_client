# Reniko Minecraft Modpack

## Требования

* Minecraft Java Edition
* Версия Minecraft: 1.21.1
* Версия NeoForge: 21.1.230
* Минимум RAM: 6 GB
* Рекомендуется: 8 GB

---

# Установка

## 1. Установите Java 21

Скачайте и установите Java 21 с официального сайта.

После установки желательно перезагрузить ПК.

---

## 2. Установите NeoForge 21.1.230

Скачайте установщик NeoForge для Minecraft 1.21.1, версия сборщика 230.

При установке выберите:

* Client

---

## 3. Один раз запустите Minecraft

Запустите профиль NeoForge один раз и закройте игру.

Это создаст нужные папки.

---

## 4. Скачайте модпак

Перейдите во вкладку справа:

* Releases

Скачайте:

* reniko_client_v0.1.zip

---

## 5. Откройте папку Minecraft

Нажмите:
WIN + R

Введите:
%appdata%/.minecraft

Нажмите Enter.

---

## 6. Установите модпак

Распакуйте содержимое архива(папку mods) в:
.minecraft/

Если Windows спросит:

* заменить файлы → Да

---

## 7. Выделите память игре

Minecraft Launcher → Установки → NeoForge → Дополнительно

Рекомендуется:

* 6 GB RAM

Пример:
-Xmx6G

---

## 8. Запустите игру

Выберите профиль NeoForge и запустите Minecraft.

---

## 9. Подключитесь к серверу

IP сервера в ТГ канале Трупати:
https://t.me/+VF7pj6bD2npjNmNi

# Решение проблем

## Игра вылетает при запуске

Проверьте:

* Установлена ли Java 21
* Установлен ли NeoForge 21.1.230
* Правильно ли распакован модпак

---

## Ошибка "Mod mismatch"

У вас отличаются версии модов от сервера.

Скачайте модпак заново из Releases.

---

## Не работает голосовой чат

* Перезапустите игру
* Проверьте интернет
* Убедитесь, что антивирус или firewall не блокирует соединение

---
