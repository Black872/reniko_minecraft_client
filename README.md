

# Reniko Minecraft Modpack

## Требования

* Minecraft Java Edition
* Версия Minecraft: 1.21.1
* Версия NeoForge: 21.1.233
* Минимум RAM: 6 GB
* Рекомендуется: 8 GB

---

# Установка

## 1. Установите Java 21

Скачайте и установите Java 21 с официального сайта.

После установки желательно перезагрузить ПК.

---

## 2. Установите NeoForge 21.1.233

Скачайте установщик NeoForge для Minecraft 1.21.1, версия сборщика 233.

При установке выберите:

* Client

---

## 3. Один раз запустите Minecraft

Запустите профиль NeoForge один раз и закройте игру.

Это создаст нужные папки.

---

## 4. Скачайте модлоадер

Перейдите во вкладку справа:

* Releases

Скачайте последнюю версию клиента:

* reniko_client_v0.*.zip

---

## 5. Откройте папку Minecraft

Нажмите:
WIN + R

Введите:
%appdata%/.minecraft

Нажмите Enter.

---

## 6. Установите модлоадер
(Желательно если уже есть моды или удалить их, или перенести в другое место)

Распакуйте содержимое архива(папку mods) в:
.minecraft/

Если Windows спросит:

* заменить файлы → Да

---

## 7. Выделите память игре

Minecraft Launcher → Установки → NeoForge → Дополнительно

Рекомендуется:

* 6 GB RAM

Пример:
-Xmx6G

---

## 8. Запустите игру

Выберите профиль NeoForge 230 и запустите Minecraft.

---

## 9. Подключитесь к серверу
Первый вход

При первом подключении AutoModpack попросит fingerprint сервера.

Введите:

```
574aeb668a225301f879864b7948e9001d301ebb56aa94c2da9aa4633cfb35ee
```

После этого:

* моды скачаются автоматически,
* конфиги синхронизируются автоматически,
* игра может перезапуститься автоматически.

Рекомендуется настроить дальность прорисовки в настройках(если вызывает просадки ФПС, понизьте параметры дальности прорисовки в моде Horizon).

# Решение проблем


## Minecraft crashed / игра вылетает

### Причины:
* мало RAM,
* старая Java,
* конфликт сторонних модов.

### Решение:
* использовать Java 21,
* выделить минимум 6 GB RAM,
* удалить лишние клиентские моды.

---

## Не работает голосовой чат

* Перезапустите игру
* Проверьте интернет
* Убедитесь, что антивирус или firewall не блокирует соединение

--- 

## "Mod mismatch"

### Причина:
* локальные моды отличаются от серверных.

### Решение:
* удалить папку:
```
.minecraft/automodpack
```
и подключиться заново.

---

## AutoModpack не скачивает моды

### Проверить:
* установлен ли AutoModpack,
* правильный ли NeoForge,
* есть ли интернет/VPN/firewall блокировки.

---

## Сервер не виден / timeout

### Возможные причины:
* сервер оффлайн,
* обновление сервера,
* проблемы DNS.

### Решение:

* подождать несколько минут и попробовать снова.
* написать в тг

---

## "Wrong Java version"

### Решение:

* установить Java 21.
